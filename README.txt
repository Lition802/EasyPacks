欢迎使用EasyPacks

作者Lition

可以在Minebbs找到我

[https://www.minebbs.com/members/lition.15796/]

本程序旨在帮助广大腐竹安装addons到服务器中

首先把您的addons后缀改为zip并解压

会得到两个文件夹

然后在您的地图文件夹里面创建behavior_packs和resource_packs两个文件夹

注意这两个文件夹与db文件夹同级

然后打开解压出来的文件夹

查看manifest.json项中的modules下的type

如果type为data则此文件夹放到behavior_packs文件夹

如果type为resource则此文件夹放到resource_packs文件夹

一切准备就绪后,运行本程序

当出现 ```[EasyPacks] json file generated successfully！```的字样时,代表json文件生成成功

注意:如果地图没有开启实验玩法的话,某些addon可能会装载失败

json文件生成之后就可以开服了!
